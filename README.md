# 📦 regex‑fa

مجموعه‌ای از Regex های آماده برای اعتبارسنجی داده‌های رایج در پروژه‌های فارسی و انگلیسی:

✅ کد ملی  
✅ شماره موبایل ایرانی  
✅ ایمیل  
✅ تاریخ و ساعت  
✅ شماره کارت، شبا و حساب  
✅ پسورد با سطوح امنیتی مختلف  
✅ آی‌پی (IPv4 و IPv6)  
✅ نام کاربری (Username)  
✅ کلمات فارسی (با انواع مختلف)  
و ...

---

## 🚀 نصب

```bash
npm install regex-fa
